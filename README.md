# Pryvit! 

Ce klaviatura dlja MacOS zasnovana na projekti ukrajins'koji latynky ščo bazujet'sja na ideji [Maksyma Prudeusa](https://ukr-latynka.org). 

# Sposib instaljaciji

Vykonaty komandu

```bash
git clone ...
```

Do dyrektoriji

```bash
cd ~/Library/Keyboard\ Layouts
```
